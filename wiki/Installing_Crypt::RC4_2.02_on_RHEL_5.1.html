Installing Crypt::RC4 2.02 on RHEL 5.1Edit

Created 2008-03-27, updated 2020-03-12

wget http://search.cpan.org/CPAN/authors/id/S/SI/SIFUKURT/Crypt-RC4-2.02.tar.gz

tar xzvf Crypt-RC4-2.02.tar.gz
cd Crypt-RC4-2.02
perl Makefile.PL
make test
sudo make install
