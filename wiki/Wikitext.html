WikitextEdit

Created 2007-06-26, updated 2024-03-03

Wikitext is an easily editable, plaintext markup language for producing HTML output. It is used as the source text format for popular wiki software such as MediaWiki.

I've written a wikitext translator for use in this website. See:

  • Official Wikitext product page
  • A quick-reference "cheatsheet" showing a subset of the available markup and output
  • rubygems.org page: http://rubygems.org/gems/wikitext
  • RubyForge project page: http://rubyforge.org/projects/wikitext (no longer updated)
  • Git repository browsers:
    • integrated repo browser
