Installing the Ragel TextMate bundle using SubversionEdit

Created 2007-04-13, updated 2017-05-18

Installation

cd Library/Application\ Support/TextMate/Bundles
svn co http://macromates.com/svn/Bundles/trunk/Bundles/Ragel.tmbundle

Output

A    Ragel.tmbundle/Preferences
A    Ragel.tmbundle/Preferences/Comments.tmPreferences
A    Ragel.tmbundle/info.plist
A    Ragel.tmbundle/README
A    Ragel.tmbundle/Syntaxes
A    Ragel.tmbundle/Syntaxes/Ragel.tmLanguage
Checked out revision 6984.

Reloading

Quit and relaunch TextMate, if it is running (choosing "Reload Bundles" from the "Bundle Editor" submenu in the "Bundles" menu is not sufficient if this is the first time you've installed the bundle).
