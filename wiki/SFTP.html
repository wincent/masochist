SFTPEdit

Created 2007-04-20, updated 2017-05-18

SFTP, the SSH file transfer protocol, is a secure transport mechanism that has largely replaced FTP for the purposes of authenticated transfer and file manipulation (FTP continues to be popular for anonymous, unauthenticated, unencrypted transfers).
