Installing Ragel 6.7 on Mac OS X 10.6.8 Snow LeopardEdit

Created 2011-05-28, updated 2020-03-12
$ curl -O http://www.complang.org/ragel/ragel-6.7.tar.gz -O http://www.complang.org/ragel/ragel-6.7.tar.gz.asc
$ gpg --verify ragel-6.7.tar.gz.asc
$ tar xzf ragel-6.7.tar.gz
$ cd ragel-6.7
$ ./configure
$ make && make test
$ (cd test && ./runtests)
$ sudo make install
