Updating to ruby-debug 0.8 using RubyGemsEdit

Created 3/19/2007, updated 5/18/2017

Updating is easy:

sudo gem install ruby-debug

Or to update all installed gems at once:

sudo gem update

Then updating the FastRI index (if you have it installed):

fastri-server -b
