ParseTree 1.7.1

Edit
Created 6/6/2007, updated 3/12/2020

The new version of ParseTree, ParseTree 1.7.1, was released on 5 June 2007.

Changes

  • 3 bug fixes:
    • Add zentest as a heckle dependency. Closes #10996
    • Fixed heckling of call with blocks.
    • Fix test_unit_heckler's test_pass? so it returns the result of the run rather than ARGV.clear
