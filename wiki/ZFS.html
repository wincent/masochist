ZFSEdit

Created 2007-11-15, updated 2017-10-14

ZFS or the Zettabyte File System is a revolutionary file-system made by Sun Microsystems and ported by Apple to Mac OS X for Leopard. At the time of writing, it has never enjoyed "official" status as a supported read-write file system for Mac OS X, and the official project page indicated that the project was discontinued in October 2009. Speculation indicates that this might have occurred due to licensing difficulties.
