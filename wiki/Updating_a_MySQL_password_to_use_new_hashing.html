Updating a MySQL password to use new hashing

Created 7/23/2013, updated 3/12/2020
  • mysql
  • wiki

SET SESSION old_passwords=0; SET PASSWORD FOR 'foo'@'bar' = password('blah'); FLUSH PRIVILEGES;
