Installing MySQL 5.0.92 on Mac OS X 10.5.8 LeopardEdit

Created 2011-02-10, updated 2020-03-12
$ curl -O http://mysql.nfsi.pt/Downloads/MySQL-5.0/mysql-5.0.92.tar.gz
$ tar xzvf mysql-5.0.92.tar.gz
$ cd mysql-5.0.92
$ ./configure
$  make
$  make test
$  sudo make install
