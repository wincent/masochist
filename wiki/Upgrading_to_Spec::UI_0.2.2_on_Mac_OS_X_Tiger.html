Future upgrade notes

Note that with Spec::UI 0.2.2 I decided for the first time to try installing Spec::UI as a Rails plug-in (see "Upgrading to RSpec 1.0.0 on Mac OS X Tiger"). Official word is that for this to actually work it "would need some minor changes in order to work smoothly". Eventually, this will mean that in addition to updating the gem for future releases, I will have to remember to upgrade copies of the plug-in installed inside Rails applications.
