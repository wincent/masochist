Updating to JRuby 1.5.2 on Mac OS X 10.6.4 Snow Leopard

Edit
Created 9/11/2010, updated 10/14/2017
$ wget http://jruby.org.s3.amazonaws.com/downloads/1.5.2/jruby-bin-1.5.2.tar.gz
$ sudo tar xzvf jruby-bin-1.5.2.tar.gz -C /usr/local                            # extract to /usr/local/
$ cd /usr/local
$ sudo rm jruby                                                                 # remove old symlink from initial install
$ sudo ln -s jruby-1.5.2 jruby                                                  # set up new symlink
$ jruby --version
# jruby -S gem update system
# jruby -S gem install bundler

As noted in my initial installation notes, in order to use JRuby installed at this location you must ensure that /usr/local/jruby/bin is in your PATH.
