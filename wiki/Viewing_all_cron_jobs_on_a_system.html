Viewing all cron jobs on a systemEdit

Created 2009-07-03, updated 2017-05-18

Red Hat Enterprise Linux

As root:

# ls -l /var/spool/cron/*
# cat /var/spool/cron/*

You'll also want to inspect:

  • /etc/crontab

And the contents of:

  • /etc/cron.d/
  • /etc/cron.daily
  • /etc/cron.hourly
  • /etc/cron.monthly
  • /etc/cron.weekly

For this purpose find might come in handy:

# find /etc -type f -path '/etc/cron*'
# find /etc -type f -path '/etc/cron*' -exec cat {} \;
