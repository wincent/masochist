Upgrading to rcov 0.8.1.2.0 using RubyGems on Mac OS X Leopard 10.5.1Edit

Created 2008-02-10, updated 2017-05-18

These notes were made while upgrading from rcov 0.8.1.0 to rcov 0.8.1.2.0 on Mac OS X Leopard 10.5.1 via RubyGems. This upgrade was necessary in order to get the pre_commit Rake task working again in the current RSpec trunk (post-1.1.3 release).
