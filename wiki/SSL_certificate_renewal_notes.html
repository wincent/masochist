SSL certificate renewal notesEdit

Created 2009-01-22, updated 2024-03-12

Note: I'm no longer maintaining these renewal notes, as I now have the whole process pretty much automated via Ansible (I just run a script to generate a new CSR, then dump the signed certificate from the CA into my Ansible configs repo and tell Ansible to install it).

Certificate authorities

  • Free certificates (and some paid options): http://cert.startcom.org/
  • Cheapest paid certificates:
    • http://www.servertastic.com/
    • http://namecheap.com/
  • Cheapest SAN certificates:
    • http://certificatesforexchange.com/
