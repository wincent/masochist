Eliminating "dyld: shared cache file is corrupt" warnings on Mac OS X LeopardEdit

Created 2008-01-08, updated 2017-10-14

Just found this gem describing how to eliminate the annoying dyld warnings that "shared cached file is corrupt" (/var/db/dyld/dyld_shared_cache_ppc):

sudo update_dyld_shared_cache -force
