Upgrading to RubyInline 3.6.7 on Mac OS X 10.5.2 LeopardEdit

Created 3/20/2008, updated 10/14/2017

These notes were made while upgrading to RubyInline 3.6.7 on Mac OS X Leopard 10.5.2.

Installing

sudo gem install RubyInline

Output

Successfully installed RubyInline-3.6.7
1 gem installed
Installing ri documentation for RubyInline-3.6.7...
Installing RDoc documentation for RubyInline-3.6.7...

Updating the FastRI index

fastri-server -b

Output

Building index.
Indexed:
* 7733 methods
* 1368 classes/modules
Needed 2.666181 seconds
