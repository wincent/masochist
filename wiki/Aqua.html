Aqua

Created 3/25/2007, updated 10/14/2017
  • macos
  • wiki

Aqua is the name of the distinctive Graphical User Interface developed by Apple for Mac OS X. It is an umbrella term that covers a large number of technologies which produce the "look and feel" of the menus, windows, icons and so forth in Mac OS X.
