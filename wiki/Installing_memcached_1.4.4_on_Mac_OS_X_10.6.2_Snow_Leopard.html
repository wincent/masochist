Installing memcached 1.4.4 on Mac OS X 10.6.2 Snow Leopard

Edit
Created 3/21/2010, updated 5/18/2017

Note: these notes show the use of wget and gpg which are not standard parts of Mac OS X. See "Installing wget 1.12 on Mac OS X 10.6.2 Snow Leopard" and "Installing GPG 1.4.10 on Mac OS X 10.6.2 Snow Leopard" for details.

libevent

Libevent is a prerequisite of memcached so we install it first:

$ wget http://www.monkey.org/~provos/libevent-1.4.13-stable.tar.gz \
       http://www.monkey.org/~provos/libevent-1.4.13-stable.tar.gz.sig
$ gpg --recv-keys --keyserver keys.gnupg.net 0A2F87E5
$ gpg --verify libevent-1.4.13-stable.tar.gz.sig libevent-1.4.13-stable.tar.gz
$ tar xzvf libevent-1.4.13-stable.tar.gz
$ cd libevent-1.4.13-stable
$ ./configure
$ make
$ make verify
$ sudo make install

memcached

$ wget http://memcached.googlecode.com/files/memcached-1.4.4.tar.gz
$ tar xzvf memcached-1.4.4.tar.gz
$ cd memcached-1.4.4
$ ./configure
$ make
$ make test
$ sudo make install
