ievmsEdit

Created 2013-10-08, updated 2017-05-18

ievms is an open source project that can download and configure the free VM images that Microsoft provides for the purposes of testing IE.

It's a great project. This free tool could help you replace a paid service like www.browserstack.com; if it does, consider making a donation.

For an example usage, see "Testing web pages in IE with ievms and VirtualBox on OS X".
