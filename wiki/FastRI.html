FastRIEdit

Created 2007-04-05, updated 2017-05-18

FastRI is a faster replacement for ri which produces colorized output. It can be installed as follows:

sudo gem install fastri

The index can then be generated (or updated) using:

fastri-server -b

Local queries can then be run using commands like this:

qri String#chomp!

Instead of the standard:

ri String#chomp!

Note that in order to search the built-in classes you must explicitly install the documentation for them as described in "Getting ri working for built-in classes".
