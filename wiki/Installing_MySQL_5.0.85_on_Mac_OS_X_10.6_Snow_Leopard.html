Installing MySQL 5.0.85 on Mac OS X 10.6 Snow Leopard

Edit
Created 8/31/2009, updated 3/12/2020

The official downloads page doesn't yet list any binary distributions for Snow Leopard, so before trying to install a Leopard binary distribution I will see if it builds cleanly from source.

$ curl -O http://mysql.rediris.es/Downloads/MySQL-5.0/mysql-5.0.85.tar.gz
$ tar xzvf mysql-5.0.85.tar.gz
$ cd mysql-5.0.85
$ ./configure --prefix=/usr/local/mysql
$ make
$ make test

So the bug in question was supposedly fixed in 5.0.83, but the test claims that it wasn't. make obviously doesn't understand the --force option, so I can't re-run.

Seeing as this is not the production server and only a development workstation, will try installing anyway and see if it works:

$ sudo make install
$ cd /usr/local/mysql
$ sudo chown -R mysql:mysql .
$ sudo bin/mysql_install_db --user=mysql
$ sudo chown -R root:wheel .
$ sudo chown -R mysql:mysql var
$ sudo -u mysql -b bin/mysqld_safe
$ mysql

At first I was not sure if this actually worked properly because of warnings issued on running mysql_install_db:
