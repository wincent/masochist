Using Awk to show only old or even lines in a fileEdit

Created 2012-07-20, updated 2017-05-18

You can use this sort of trick, combined with grep, to de-dupe repeated lines in a log file:

$ awk 'NR%2==0' file
$ awk 'NR%2==1' file
