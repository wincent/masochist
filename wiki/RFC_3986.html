RFC 3986Edit

Created 2007-06-05, updated 2017-05-18

RFC 3986 is an RFC titled, "Uniform Resource Identifier (URI): Generic Syntax". A grammar is provided that describes the syntax; it is written using ABNF notation, which is itself described in RFC 2234.
