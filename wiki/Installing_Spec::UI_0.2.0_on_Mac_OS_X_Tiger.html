Installing Spec::UI 0.2.0 on Mac OS X TigerEdit

Created 2007-05-18, updated 2017-05-18

Spec::UI version 0.2.0 was released alongside Rspec version 0.9.4. These notes were made while installing it for the first time.

sudo gem install spec_ui
