Installing Redis 2.2.9 on Mac OS X 10.6.7 Snow LeopardEdit

Created 2011-06-12, updated 2020-03-12
$ curl -O http://redis.googlecode.com/files/redis-2.2.9.tar.gz
$ tar xzf redis-2.2.9.tar.gz
$ cd redis-2.2.9
$ cd src
$ make
$ make test
$ sudo make install
