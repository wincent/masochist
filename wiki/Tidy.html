TidyEdit

Created 2007-04-12, updated 2017-05-18

Tidy, also commonly known as HTML Tidy, is a tool for correcting malformed HTML and/or applying uniform formatting to that HTML.

The tidy command line tool is shipped as part of the standard Mac OS X distribution. Type man tidy at the Terminal for more information.
