Upgrading from rsync 2.6.9pre2 to 2.6.9Edit

Created 2006-11-07, updated 2017-05-18

Notes made in the upgrade from rsync 2.6.9pre2 to 2.6.9:

wget http://rsync.samba.org/ftp/rsync/rsync-2.6.9.tar.gz
tar zxvf rsync-2.6.9.tar.gz
cd rsync-2.6.9
./configure && make
sudo make check
sudo make install

I then did a test remote backup using my rsync-backup.sh script and everything is confirmed to be working.
