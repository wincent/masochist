Installing heckle 1.4.1 on Mac OS X 10.5.1 LeopardEdit

Created 2008-02-09, updated 2017-05-18

These notes were made while installing heckle 1.4.1 on Mac OS X 10.5.1 Leopard. Heckle is a prerequisite for running the Rspec pre_commit Rake task (required in order to submit patches).

Installation

$ sudo gem install heckle
Successfully installed heckle-1.4.1
1 gem installed
Installing ri documentation for heckle-1.4.1...
Installing RDoc documentation for heckle-1.4.1...
