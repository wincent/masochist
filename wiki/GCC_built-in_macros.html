GCC built-in macrosEdit

Created 2006-11-01, updated 2020-03-12

For Objective-C code

gcc -dM -E -x objective-c - <<<''

For C code

gcc -dM -E -x c - <<<''

or:

:| gcc -dM -E -x c -

or:

touch empty.c
cc empty.c -E -dM -o - | sort
