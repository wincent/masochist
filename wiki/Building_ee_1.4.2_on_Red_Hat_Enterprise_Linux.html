Building ee 1.4.2 on Red Hat Enterprise LinuxEdit

Created 10/16/2006, updated 5/18/2017

Notes made while building and installing ee version 1.4.2 on Red Hat Enterprise Linux ES 3:

wget http://www.users.qwest.net/~hmahon/sources/ee-1.4.2.src.tgz
tar xzvf ee-1.4.2.src.tgz
cd easyedit
make
./ee
sudo make install
