Buildtools

Edit
Created 9/9/2006, updated 3/12/2024

The Wincent Build Tools, or simply "buildtools", are a collection of helpful scripts and configuration files that simplify some build tasks when working with Xcode. All Wincent products make use of the buildtools. The official project page is:

http://wincent.dev/a/products/buildtools/
