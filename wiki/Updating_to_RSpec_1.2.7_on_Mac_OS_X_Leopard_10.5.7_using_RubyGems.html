Updating to RSpec 1.2.7 on Mac OS X Leopard 10.5.7 using RubyGems

Created 6/23/2009, updated 5/18/2017
  • leopard
  • rspec
  • updates
  • wiki

Installing

Given that the rspec-rails gem depends on the rspec gem, you can install both by doing sudo gem install rspec-rails:

$ sudo gem install rspec-rails
