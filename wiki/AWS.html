AWS

Edit
Created 11/17/2009, updated 5/18/2017

Amazon Web Services, or AWS, is a collection of network-based services sold by Amazon. The services enable a variety of things like hosting websites, computation, storage, using Amazon's infrastructure.

See also

  • S3 (Simple Storage Service)
  • EC2 (Elastic Compute Cloud)
  • EBS (Elastic Block Store)
  • Amazon Linux AMI
