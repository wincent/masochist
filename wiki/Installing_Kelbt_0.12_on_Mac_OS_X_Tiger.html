Installing Kelbt 0.12 on Mac OS X Tiger

Edit
Created 5/23/2007, updated 3/12/2020

While upgrading from Ragel 5.20 to 5.21 on Mac OS X Tiger I decided to try upgrading to a the new version of Kelbt as well, Kelbt 0.12.

Unfortunately, the tests did not pass, so I decided not to proceed with the install.

Building

wget http://www.cs.queensu.ca/~thurston/kelbt/kelbt-0.12.tar.gz
tar xzvf kelbt-0.12.tar.gz
cd kelbt-0.12/
./configure
make
cd test
./runtests
