Upgrading to Spec::UI 0.2.3 on Mac OS X TigerEdit

Created 2007-05-24, updated 2017-05-18

Release notes

The official RSpec 1.0.1 release notes don't make any specific mention of changes in Spec::UI 0.2.3.

Future upgrade notes

I'd like to be able to install Spec::UI as a Rails plug-in but I haven't yet heard any update to the official word that this "would need some minor changes in order to work smoothly".
