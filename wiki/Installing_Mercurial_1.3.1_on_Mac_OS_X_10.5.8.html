Installing Mercurial 1.3.1 on Mac OS X 10.5.8

Edit
Created 8/21/2009, updated 3/12/2020
$ wget http://mercurial.selenic.com/release/mercurial-1.3.1.tar.gz
$ tar xzvf mercurial-1.3.1.tar.gz
$ cd mercurial-1.3.1
$ make all
$ sudo make install
$ export PYTHONPATH=/usr/local/lib/python2.5/site-packages
