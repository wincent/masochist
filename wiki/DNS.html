DNSEdit

Created 2008-03-27, updated 2024-03-12

DNS, or the Domain Name System, is infrastructure that enables clients on a network to translate human-readable symbolic addresses like "wincent.dev" to their numeric IP addresses (209.235.192.161). DNS is often also used as a short hand to refer to an specific, individual Domain Name Server (that is "the DNS" rather than "the DNS server").

In November 2009 I started using Nettica as an external DNS provider for this site.
