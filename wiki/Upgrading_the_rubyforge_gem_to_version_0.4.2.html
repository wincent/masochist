Upgrading the rubyforge gem to version 0.4.2Edit

Created 2007-05-25, updated 2017-05-18

These notes were made while upgrading to rubyforge 0.4.2, the new release of the gem that provides a programmatic Ruby interface to RubyForge.

sudo gem install rubyforge
