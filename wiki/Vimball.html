Vimball

Created 3/21/2010, updated 5/18/2017
  • vim
  • vimball
  • wiki

Vimballs are files in the "vimball" format which are intended to make installing and managing Vim plugins easy.

The format was created by Charles Campbell, and the plugin for working with vimballs is available from here. The plugin is included with all recent versions of Vim.
