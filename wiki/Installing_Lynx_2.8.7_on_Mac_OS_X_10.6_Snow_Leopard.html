Installing Lynx 2.8.7 on Mac OS X 10.6 Snow LeopardEdit

Created 2009-09-04, updated 2020-03-12
$ curl -O http://lynx.isc.org/lynx2.8.7/lynx2.8.7.tar.bz2
$ tar xjvf lynx2.8.7.tar.bz2
$ cd lynx2-8-7
$ ./configure --with-ssl && make
$ sudo make install
