# Verification

Before committing, verify changes by running:

1. `cargo check` — confirm the code compiles
2. `bin/test` — run the test suite
3. `bin/check-format` — check formatting
4. `bin/clippy` — run lint checks

# Planning and memory

Use the `.claude-notes/` directory for storing project-specific memory files. Depending on the size of the initiative, your user will prompt you to read and/or create planning and progress documents in this directory (which is in `.gitignore`).
