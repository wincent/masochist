Site updates

Created 2009-04-24, updated 2024-03-055 minute read

I just deployed an updated version of the site so if you notice any anomalies please let me know (either by opening a ticket or emailing me{.mailto}). A full list of changes appears below.

Changes

Most of these a code-level changes with little impact at the user-interface level for public visitors (although for admin users there are quite a few improvements, especially on the AJAX side of things).

Some noteworthy changes for public visitors are:

  • You can now open tickets with only a summary (no description). The intention here is to make the issue tracker much easier to use as a super-easy-and-convenient "TODO" list; I'm hoping that users won't abuse the feature and file bug reports or feature requests without the necessary details.
  • I've jettisoned Prototype and Scriptaculous in favour of jQuery, which means a lot less JavaScript has to be downloaded on the first page-view and should result in much better load times.
  • A couple of bugs related to "authenticity tokens" being inappropriately included in the search form were fixed:
    • The search form on static error pages like the 404 (not found) and 500 (internal server error) pages now works.
    • "Authenticity tokens" are no longer leaked onto page-cached pages, effectively fixing likely breakage of the search form on all such pages.
