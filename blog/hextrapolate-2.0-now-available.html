Hextrapolate 2.0 now available

Created 2007-04-23, updated 2026-03-141 minute read

Following a successful beta testing period the new version of Hextrapolate (formerly WinHex), 2.0, is now available.

This is a ground-up rewrite of the original WinHex number conversion utility with many new features, bug fixes and enhancements. This is also the first Universal Binary release for Hextrapolate. See the full changelog for all the details. The name-change took place to avoid confusion with a Windows product of the same name.

Hextrapolate licenses can be purchased here, and you can download the product here.
