Wincent Icon Utility 2.0.1 released

Created 2009-08-29, updated 2026-03-161 minute read

I've just published a new version of Wincent Icon Utility, version 2.0.1, especially for Mac OS X 10.6 Snow Leopard. Wincent Icon Utility is a small command line tool for taking an ".icns" icons file and applying it to a directory as a custom folder icon.
