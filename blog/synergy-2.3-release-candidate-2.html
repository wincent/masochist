Synergy 2.3 release candidate 2

Created 2005-11-30, updated 2026-03-141 minute read

Release candidate 2 for Synergy 2.3 has just been released. The new candidate fixes a problem with license code validation on the Intel platform caused by byte-ordering issues. A full changelog appears here.

The release has already been seeded to beta testers. If you wish to participate in the testing effort you can download the release candidate from here and report any bugs or issues that you might find. The current, stable release (version 2.2) continues to be available for download from here. If you're interested in getting access to this kind of build in the future subscribe to the synergy-beta mailing list. If all goes well, this build will shortly be declared final and a formal release announcement will be made; if you'd like to be among the first to know, subscribe to the synergy-announce or wincent-announce mailing lists.
