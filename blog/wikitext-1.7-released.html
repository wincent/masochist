Wikitext 1.7 released

Created 5/12/2009, updated 3/4/20241 minute read

A new version of the Wikitext gem, version 1.7, is now available. It contains a couple of bugfixes and a sizeable rewrite of the internals leading to a big performance boost (double the speed or more in many cases). See the high-level or low-level release notes for more details.

Wikitext is a fast wikitext-to-HTML translator with a syntax very close to that used by MediaWiki. It is a Ruby extension written with speed and robustness in mind using a Ragel-generated scanner and a hand-coded parser, all in C. It has an enormous spec suite.

You can browse the source code repo, or download the gem from the RubyForge downloads page. For an overview of how to use the extension check out the RDoc here. If you have RubyGems then you can install it with sudo gem install wikitext (although it may take some time before the new version has propagated out to all the gem servers).

If you find the extension useful please consider making a donation via PayPal.
