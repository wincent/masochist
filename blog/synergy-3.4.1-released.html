Synergy 3.4.1 released

Created 2009-09-09, updated 2026-03-161 minute read

I've just prepared an update to the Synergy 3.4 series, version 3.4.1, whose sole purpose is to backport a bugfix (see ticket #1383) for the benefit of users who don't want to or can't upgrade to the 4.0 series yet, but still want to run on Mac OS X 10.6 Snow Leopard.

If you are a 3.4 user and aren't using Snow Leopard yet then there is absolutely no reason to update to 3.4.1.

Synergy is a lightweight, highly-customizable iTunes controller. You can download the new release from the download page and purchase a license using this page. The Synergy product pages were recently completely revamped for 4.0, so be sure to check them out.
