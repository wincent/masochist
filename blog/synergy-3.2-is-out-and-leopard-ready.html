Synergy 3.2 is out and Leopard-ready

Created 10/24/2007, updated 3/12/20241 minute read

Synergy 3.2 is now available for immediate download. This is the final compatibility release in preparation for the Leopard launch in a couple days' time. A full changelog appears here. This version of Synergy should run on all versions of Mac OS X from Jaguar right through to Leopard.

You can download the new release from here and purchase a license using this page. This is a free upgrade for users who already hold a valid Synergy license.
