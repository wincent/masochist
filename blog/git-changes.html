Git changes

Created 2007-07-11, updated 2024-03-121 minute read

I've started adding more Git content to the wiki, and the more I explore, the more impressed I am.

Meanwhile, using Git becomes more and more palatable.

My sense of aesthetics makes me feel queasy about typing commands like these:

git-tag -l
git-commit

But I am quite happy to type:

git tag -l
git commit
