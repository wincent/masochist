Synergy 3.2.2 released

Created 2009-04-02, updated 2026-03-141 minute read

I'm pleased to announce the availability of the latest release in the Synergy stable 3.2 series (version 3.2.2). A full changelog appears here.

You can download the new release from the download page (direct link) and purchase a license using this page.

How does this release compare to the build given away with MacHeist "Giving Tree" promotion?

Both this release (3.2.2) and the MacHeist giveaway (labeled as version 3.3) derive from the same codebase. The only difference between them is that the MacHeist giveaway build does not require a serial number.
