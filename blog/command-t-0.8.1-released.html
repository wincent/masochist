Command-T 0.8.1 released

Created 2010-09-14, updated 2024-03-031 minute read

I've just released version 0.8.1 of the Command-T plug-in for Vim. This is a powerful, open source file-navigation plug-in inspired by the "Command-T" feature in TextMate.

This is a minor maintenance release containing documentation updates and one bugfix (fixing possible mapping issues for users with set notimeout in their ~/.vimrc). A full change-log appears under HISTORY in the documentation.

If you're a Vim user check out the screencasts and give the plug-in a try. If you'd like to support development you can use the donations page to make a donation, or consider submitting a patch for the project.
