Wikitext 1.8 released

Created 2009-05-27, updated 2024-03-041 minute read

A new version of the Wikitext gem, version 1.8, is now available. It adds one new feature — the ability to specify a CSS class on <pre></pre> blocks so that they can be targeted by JavaScript, or other, syntax-highliters — and some performance enhancements. See the high-level or low-level release notes for more details.

Wikitext is a fast wikitext-to-HTML translator with a syntax very close to that used by MediaWiki. It is a Ruby extension written with speed and robustness in mind using a Ragel-generated scanner and a hand-coded parser, all in C. It has an enormous spec suite.

You can browse the source code repo, or download the gem from the RubyForge downloads page. For an overview of how to use the extension check out the RDoc here. If you have RubyGems then you can install it with sudo gem install wikitext.

If you find the extension useful please consider making a donation via PayPal.
