nginx 0.7 series to become the new "stable" release

Created 2009-05-20, updated 2017-05-181 minute read

Igor Sysoev, the maintainer of nginx, has just posted to the mailing list:

I'm going to branch 0.8 development version and mark 0.7 as stable on this or the next week.

So looks like the jump from the 0.6 series will be in order soon.
