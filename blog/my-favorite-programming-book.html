My favorite programming book

Created 2006-01-27, updated 2020-03-121 minute read

My favorite programming text of all time is the O'Reilly C Pocket Reference by Peter Prinz and Ulla Kirch-Prinz:

If only more books could be written like this. Small, compact, clear, comprehensive, it tells me everything I've ever needed to know about C. It costs a mere $9.95 and is worth every penny. Nowadays, in this age of online documentation, the O'Reilly Pocket Reference Series is about the only printed material that I ever access with any frequency.
