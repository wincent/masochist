Wincent Icon Utility 2.0.2 released

Created 2010-11-02, updated 2024-03-041 minute read

Version 2.0.2 of Wincent Icon Utility is now available. Wincent Icon Utility is a small command line tool for Mac OS X 10.6 Snow Leopard which can be used to take an .icns icons file and apply it to a directory as a custom folder icon.

Wincent Icon Utility is open source code released under the GPL. If you find it useful please consider making a donation.
