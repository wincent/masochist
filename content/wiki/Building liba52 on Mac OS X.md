These notes were made while building version 0.7.4 of [liba52](/wiki/liba52) on [Mac OS X](/wiki/Mac_OS_X):

    wget http://liba52.sourceforge.net/files/a52dec-0.7.4.tar.gz
    tar xzvf a52dec-0.7.4.tar.gz 
    cd a52dec-0.7.4
    ./configure 
    make
    sudo make install
