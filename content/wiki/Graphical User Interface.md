A [Graphical User Interface](/wiki/Graphical_User_Interface) typically consists of a series of windows, menus, icons and other graphical elements that a computer user interacts with not only using the keyboard but also with some kind of pointing device such as a mouse.

[Graphical User Interface](/wiki/Graphical_User_Interface) is sometimes abbreviated to [GUI](/wiki/GUI) for short.
