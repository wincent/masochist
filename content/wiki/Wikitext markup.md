[Wikitext markup](/wiki/Wikitext_markup) is my preferred form of entering text in web applications. It's used by [MediaWiki](/wiki/MediaWiki) (the software which powers this [wiki](/wiki/wiki)).

I've written my own [open source](/wiki/open_source) wikitext-to-[HTML](/wiki/HTML) converter, a [Ruby](/wiki/Ruby) extension written in [C](/wiki/C) for use in a [Rails](/wiki/Rails) application of mine. You can browse the source here:

-   <http://git.wincent.com/wikitext.git>
