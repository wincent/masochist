[RFC 3986](/wiki/RFC_3986) is an [RFC](/wiki/RFC) titled, "Uniform Resource Identifier (URI): Generic Syntax". A grammar is provided that describes the syntax; it is written using [ABNF](/wiki/ABNF) notation, which is itself described in [RFC 2234](/wiki/RFC_2234).
