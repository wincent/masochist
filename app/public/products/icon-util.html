Wincent Icon Utility

Wincent Icon Utility is a small command line tool for Mac OS X 10.6 Snow Leopard which can be used to take an .icns icons file and apply it to a directory as a custom folder icon.

  • Manual page
  • Web-based repository browser

Wincent Icon Utility is open source code released under the BSD license. If you find it useful please consider making a donation.
