Feature request #723: 'Time remaining' to show in floater

Kind feature request
Product Synergy
When Created 2008-05-17T17:43:43Z, updated 2008-05-19T03:06:11Z
Status open
Reporter anonymous
Tags no tags

Description

I have 'Show Floater' bound to a hotkey, so that if I've forgotton the name of a track I can bring it up easily. However, for various reasons I'd also find it useful to know how long the track has left to play, as really it's the only reason I need to open iTunes anymore: to check the progress bar at the top of the window. Having a feature to show the time remaining could be an advanced option to augment the 'Track Duration' text in the floater: if checked, it would display the total duration for the first ~5 seconds (for when the floater comes up at the beginning of each track) and the track's time remaining thereafter (for checking the floater with the hotkey).
