Comments

  1. Greg Hurrell 2010-11-20T10:36:19Z

    Can you please tell me where the broken download link is? It must be on an old page somewhere and I'd like to fix it.

    If I recall correctly, the Info.plist is for informative purposes only (ie. it says who the author of the button set was etc) but it's not strictly required so you can just ignore or remove that.

    As for the rest of the set, it is probably a better idea to rename your modified set so that it doesn't have the same name as the original. It may be that the original is installed (at ~/Library/Application Support/Synergy/Button Sets/reflex.synergyButtons) and when you double-click your modified set to install it, the original is not getting replaced because Synergy believes it is already installed.

    To avoid confusion, you should just be able to pick a new name (eg. reflex2.synergyButtons or whatever) and double-click it to install it in the Button Sets folder mentioned above, and Synergy should switch to show your new set immediately.
