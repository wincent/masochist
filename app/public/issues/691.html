  1. Greg Hurrell 2008-05-03T16:14:37Z

    What version are you using?

    If you are using the 3.5 experimental alpha series then you are probably seeing issue #640. I'd recommend going back to the stable 3.2 series in that case.

    As for not seeing album art, you are probably running an older version. iTunes 7.6.1 and 7.6.2 broke cover art support, and in the last release I did I implemented a workaround for the problem. See issue #668 for more details on that. The new versions are 3.5a7 (alpha series) and 3.2.1b (stable series).
