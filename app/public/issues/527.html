Feature request #527: REQUEST: Option to turn off "pause" notifications for Growl

Kind feature request
Product Synergy
When Created 2006-12-29T13:06:57Z, updated 2007-05-15T12:21:19Z
Status closed
Reporter bastian
Tags no tags

Description

when growl notifications are turned on, it displays a notification when plays a song, that is very useful, but when it displays the pause notification... we dont need to know when we pause the song, because WE paused the song.

Disabling the pause notification is a good idea or, like with all the other growl-supported applications, putting the options of notifications in the Growl pref pane is a better idea. (Example: in the synergy's growl pref. pane, the options to turn on or off the play, pause and rating notifications)
