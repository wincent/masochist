Feature request #821: Move "random" button set option from "Advanced" tab to "Menu Bar" tab

Kind feature request
Product Synergy
When Created 2008-05-28T18:10:26Z, updated 2008-05-30T06:15:20Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

I occasionally get support queries from people complaining about Synergy's menu bar buttons changing on them every time they start up (example). This is inevitably caused by them having the "random" button set rotation turned on in the "Advanced" tab of the preferences.

I think this is a UI problem. They simply don't know or forget about the option because it's out of sight when they are looking at the "Menu Bar" tab. I'll see if I can juggle things around and make room for the option in the "Menu Bar" tab.
