Bug #593: Album covers should not be fetched if one exists already

Kind bug
Product Synergy
When 2007-07-23T10:11:49Z
Status closed
Reporter Steve Lombardi
Tags no tags

Description

I would like to have a preference to merley DISPLAY existing album are, and NOT fetch any art from the internet. If no art is set for a given track, then just show the "synergy icon".

I am very diligent about all my songs in iTunes having album cover art, and having the correct art. However, when I play my tracks using Synergy, I often see Synergy has chosen a different -- and completely incorrect -- album cover.

For example I'm playing "Baby, it's cold outside" by Pezband, on the album Poptopia!, which I added cover art to, but now Synergy has selected a Donny OSmond cover!

I don't expect Synergy to be able to find some of these albums -- in fact, for alor of them (this one included) I have had to search the web manually. I just want to use Synergy as a player, and not have it attempt to find or assign album art.
