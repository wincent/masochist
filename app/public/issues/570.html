Feature request #570: REQUEST: Toggle shuffle in Synergy

Kind feature request
Product Synergy
When Created 2007-05-15T11:53:37Z, updated 2007-05-14T09:07:14Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

It'd be great to have an option for this fourth button; it's the one thing I still open itunes for. My music habits are basically: - listen on shuffle till i find something i really enjoy - turn shuffle off and listen to the rest of the album - get annoyed again and turn shuffle on

I'd love to be able to do this from synergy.

There are really three ways in which Shuffle could be turned on and off:

1. Via a hot key or hot keys 2. Via an an additional item in the Global Menu 3. Via an additional button in the row of menu bar buttons

"1" and "2" are already implemented in Synergy Advance (see http://advance.wincent.com/), so I am opening this feature request here to record interest in the feature of Synergy. I don't think option "3" is very likely for Synergy because it would require all of the existing button sets (and there are about 100 of them) to be augmented. It is a possibility for Synergy Advance, though, because the skins format is still under development.
