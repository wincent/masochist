Bug #2045: Difficulty overwriting existing hot keys

Kind bug
Product Synergy
When 2013-01-02T00:05:11Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

There appears to be a bug in the new hot key implementation in Synergy 5.0a that makes it hard or cumbersome to overwrite existing hot keys.

To repro:

  1. Set a key for next track (eg. Control-Option-Command-L)
  2. Apply
  3. Set the same key for another command (eg. "Volume up")

Observe how when pressing the key in the last step, iTunes skips to the next track instead of letting you record the new key combo.
