  1. Greg Hurrell 2008-12-29T23:08:29Z

    There's no expiry. It's a fully unlocked version, and doesn't require a serial number to be activated.

    It is compatible with Mac OS X 10.5 Leopard.

    There is a 3.5 version currently in alpha, but it's affected by an Apple bug so I don't actually recommend that people use it. (See bug #640 for more details on the actual problem.) As such the 3.5 series is effectively on hold until Apple fixes things, and I've been working on 3.2.x/3.3 instead.

    As the MacHeist promo version is a freebie, no upgrades are included.
