Only when using WinSwitch?

  1. anonymous 2004-09-26T05:29:24Z

    I was wondering if the switch-in and switch-out actions work only if you change users using the WinSwitch menu item, or if they also work when you're at the log-in screen. eg. I have my screen saver set to lock my screen, then the login window appears and I switch users, does the app/script/whatever launch?

  2. Greg Hurrell 2004-09-26T09:20:54Z

    According to the Apple documentation, only on switching in and switching out, not on logging in and logging out.

    WinSwitch uses the NSWorkspaceSessionDidResignActiveNotification and NSWorkspaceSessionDidBecomeActiveNotification.

    See Apple's NSWorkspace documentation for details on when these notifications get posted.
