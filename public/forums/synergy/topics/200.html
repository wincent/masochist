  2. heavyboots 2007-03-19T01:47:19Z

    I'm running 10.4.9 (Combo updater) on a G5 in the US with no problems.

    Here's some routine maintenance-type stuff you could try:

    [1] Check whether manually starting Synergy from the System Prefs helps. Also, is it only the menu bar or is Synergy running but not showing them? Can you use hotkeys still? Do you see the track floater?

    [2] Try repairing permissions/verifying the directory of your startup disk with Disk Utility.

    [3] Use a utility like Onyx to empty your caches (Cleaning icon->Caches tab->User/Application/Font/System caches should be checked. You'll have to restart afterwards.)

    [4] If you did the delta updater via Sofware Update, you might try downloading the Combo updater and running that manually, but that is pretty last-ditch if everything else is still working correctly, so I wouldn't do it without a full backup, etc.

    Best of luck and hope that helps...
