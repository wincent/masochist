Incompatibility between Synergy 1.7, Mac OS X 10.2

  1. Greg Hurrell 2005-03-04T21:53:40Z

    A conflict has been discovered between Mac OS X 10.2, Synergy, and Growl which prevents version 1.7 from running on those systems (regardless of whether Growl is installed). A solution is currently under investigation and a fix (1.7.1) will be released on Monday at the latest. See this report for more information. Sorry about the inconvenience.
