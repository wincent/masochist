  2. Greg Hurrell 2005-07-03T13:22:31Z

    If you just leave Synergy running it should get pretty much all the art eventually; it puts art requests in a queue and waits between requests so as not to produce too many hits on the Amazon servers.

    There is already a thread in this forum about manually adding art:

    http://wincent.com/a/support/forums/showflat.php?Number=67

    Personally though, I recommend dragging the art into iTunes, much easier and if its only a JPEG file (a 40KB graphic in a 4MB music file) it only adds 1% to the size of the file, and then you can see the art in your iPod Photo if you've got one...
