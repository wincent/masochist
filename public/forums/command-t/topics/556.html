[Feature request] option to set look-up directories

  1. anonymous 2014-02-11T08:15:25Z

    Command-T has an option: let g:CommandTWildIgnore to blacklist directories/files/patterns.

    But sometimes, whitelisting is far more effective than blacklisting. For example, an option like: let g:CommandTDir=/path/to/projects/,/path/to/tests/,/path/to/production/ sets Command-T to *only* look-up in those directories.
