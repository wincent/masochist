Bug #1355: Synergy 3.4 fails to launch at login

Kind bug
Product Synergy
When Created 2009-07-26T17:26:03Z, updated 2009-07-27T04:21:59Z
Status closed
Reporter anonymous
Tags no tags

Description

I recently upgraded to Synergy 3.4, and the program no longer launches when I login ("Launch Synergy at login") is checked in the System Preferences. I have to click the Start/Stop button every time I boot my machine in order to get the program running. This didn't happen with the previous version of Synergy I was using (3.1, I think? I can't remember).

This affects two machines - a MacBook Pro and an iMac - both running OS X 10.5.7.
