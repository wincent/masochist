Feature request #56: REQUEST: Smaller volume control intervals

Kind feature request
Product Synergy Advance
When Created 2004-12-06T17:39:41Z, updated 2005-05-03T09:37:58Z
Status closed
Reporter William Henderson
Tags no tags

Description

Applescript lets you set the iTunes volume interval to anywhere between 0 and 100. The number of intervals available in Synergy (and iTunes, for that matter) is not enough for many users, especially those using Airport Express (who cannot use System volume in combination with iTunes volume for a bit more control, and who also probably are not right next to the speakers they are controlling and cannot simply turn them down). It would be really nice to have a simple option that allowed a greater degree of control via the hotkey/floater volume control feature. Thanks.
