Feature request #1140: keeping synergy in menu bar

Kind feature request

When 2008-09-20T09:38:24Z
Status closed

Description

How do I get Synergy to stay put in my OS X menu bar? Whether I'm playing iTunes or not. Right now it keeps dropping out for no reason.

Thanks

Comments

  1. Greg Hurrell 2008-09-20T10:40:22Z

    Depends what you mean by "dropping out".

    There are three things that might be the cause:

    1. If you're running the 3.5 series on Leopard you might be crashing because of the Apple bug described in ticket #640; the solution in that case is to use the 3.2 series instead.
    2. You might have things set up in the "Menu Bar" tab of the preferences to show/hide the controls depending on whether iTunes is running
    3. If you are short on menu bar real estate and you switch between applications with different numbers of menus and different width text in those menus, Mac OS X might be temporarily hiding one or more items on the right hand side (ie. Synergy) to make room for the menus on the left hand side; in this case there's really nothing you can do except run fewer menu extras/status items, or use a bigger screen; you can also try loading Synergy before other items to make it appear further to the right.

    If you want more info about any of these possibilities feel free to add another comment. For now marking as closed.
