Feature request #231: REQUEST: Show track/playlist time in menu bar

Kind feature request
Product Synergy Advance
When Created 2005-05-10T21:28:24Z, updated 2006-07-05T02:05:36Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

"I was just going to request the ability to have the time displayed in the menu bar for either Classic or Advanced.

Here is my suggestion for the options: a) current time/total track time b) current time/total playlist time"

I have a module for showing the track time in the menu bar in Synergy Advance; you'll see the module in one of the forthcoming public preview releases.

The part of this request that is new is part (b). That one might be quite difficult to implement, although not impossible. The problem is that with very large playlists (for example, the "Library" playlist) it could take a long time to calculate the total time. I'll see what I can do though.
