Feature request #305: REQUEST: Copy current track details to clipboard or log file

Kind feature request
Product Synergy
When Created 2005-09-26T22:11:48Z, updated 2006-07-05T02:06:35Z
Status open
Reporter Hector Cabarcas
Tags no tags

Description

I would love the ability to copy the current track's song info (artist and song name) to the clipboard or directly to the desktop. There are so many times I'm listening to an Internet stream and find a song I really like. Instead of having to physically write it down on paper and then type it into a text file manually, something like this feature request would be great. Take it a step further, and provide a shortcut that would automatically ammend the current track to a text file entitled MySynergyWishList residing in a location specified in preferences.

Hello Wincent, it's Hector...the creator of 2 button sets. Hope you're doing fine.

  2. Hector Cabarcas 2005-09-26T23:52:08Z

    Hey Wincent. Nice hearing from you. Here's how it would work...

    1) I'm listening to a song and decide I really like it. 2) I hit a shortcut (preferrably) or a menu item from the menu bar. This shortcut/menu item would write the current song's Artist and Song Name to a text file called "MySynergyWishList.txt" which resides somewhere on the hard drive. You could have a preference that allows the user to decide where to place that file. 3) The next time I choose the shortcut/menu item, the next song is amended to the end of the text file--right after the last song & artist. Like what follows. Notice that the 3rd line combines the artist and song so that a person can copy and paste that line and go to the Search box in the iTunes music store.

    Artist: Theo Bishop Song: Walk With Me iTunes Search: Theo Bishop Walk With Me

    Artist: Spyro Gyra Song: Ariana iTunes Search: Sypro Gyra Ariana

    ---

    That's all I really mean. Nice and simple. This would be a huge feature in my mind. Let me know what you think.
