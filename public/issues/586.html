Feature request #586: REQUEST: stop-after-current-track

Kind feature request
Product Synergy
When Created 2007-06-20T18:32:15Z, updated 2007-06-13T05:12:55Z
Status open
Reporter Ian Barland
Tags no tags

Description

A hotkey for stop-after-the-current-track-finishes would be great. I often want to finish out the current song before stepping away from my machine.

To avoid needing new buttons or hotkeys, you could have an option: "pressing stop once => stop after current; pressing stop twice => actually stop."

(Fwiw, googling 'itunes "stop after current"' gives about 350 hits, mostly blogs asking for this feature or wistfully recalling an early winamp version which did that. I suspect this one feature would be a major advantage over competitors.)

A related feature would be "upon a stop-request, fade out over 1s instead". (Er, min(1sec, time-remaining).)

Thx for a great product, --Ian
