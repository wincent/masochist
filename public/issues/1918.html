Bug #1918: Serial number invalid but can't input new one after purcshasing.

Kind bug
Product Synergy
When 2011-11-17T02:18:56Z
Status closed
Reporter anonymous
Tags no tags

Description

I upgraded to Synergy 4.5.2 and every time I restart a window pops up to say that my serial number is not valid for this version. So I bought a new one and the software will not take the new serial number to stop the window from popping up.

Comments

  1. Greg Hurrell 2011-11-17T02:22:47Z

    You should be able to remove the old registration info by hitting the "Unregister" button in the "About" tab of the Synergy Preferences. At that point you should be able to enter the new license info.

    If that doesn't work, it is possible that your preferences file has become corrupted. You could try deleting it. (It can be found at /Users/<your user name>/Library/Preferences/org.wincent.Synergy.plist.)
