  2. Greg Hurrell 2009-02-09T00:49:19Z

    As you noted, the copy you received via MacHeist is fully-unlocked and free. Being free, it does not include any updates.

    But please note that the MacHeist version is the latest released version, so whatever version you installed, it is not an "upgrade", so I'd advise you to go back to the MacHeist promo version.

    Perhaps you installed a build from the 3.5 alpha series. If you look at the version history you'll see that it's not actually the latest release despite the higher version number. The 3.5 series is a rewrite using Garbage Collection on Leopard, but due to an Apple bug for which there is no work around the series is effectively on hold until Apple fixes the bug (see ticket #640 for all the details).

    So I haven't done any work at all on the 3.5 series for some time now, and my latest work has all been on the 3.2 series, and then on 3.3 (the MacHeist release).

    Apple is aware of the bug and has stated their intention to fix it, but they are acting quite slowly on it. I personally doubt that they'll fix it prior to Snow Leopard.
