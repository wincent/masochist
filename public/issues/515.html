Feature request #515: REQUEST: Show current track info in Skype

Kind feature request
Product Synergy Advance
When Created 2006-12-04T06:20:43Z, updated 2007-09-19T14:39:15Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

"It would be nice if you could display the current track in Skype. Same features like iChat are possible. I use both, would be fun if it worked for Skype too :)"

Thanks for submitting the feature request. This kind of thing is exactly the reason why Synergy Advance uses a plug-in based architecture; the iChat support is provided in the form of a plug-in and in theory Skype support could take the form of a plug-in as well. I say "in theory" because I haven't looked into Skype's AppleScript support yet. Skype will need to support the ability to set the status message via AppleScript in order for Synergy Advance to communicate with it.

Will mark this as ASSIGNED. Please add yourself to this feature request as a "CC" if you'd like to keep informed.
