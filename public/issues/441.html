Bug #441: Typo in opening paragraph

Kind bug
Product WinSwitch
When 2006-07-08T05:32:39Z
Status closed
Reporter Christopher Cormack
Tags no tags

Description

In the last sentence of the first paragraph of the WinSwitch product information page, you have the sentence:

"WinSwitch fixes that and brings a number of other powerful features that are not available on other Panther or Tiger (Mac OS X 10.4)."

I think that the "other" should be "either", so:

"WinSwitch fixes that and brings a number of other powerful features that are not available on either Panther or Tiger (Mac OS X 10.4)"
