Feature request #89: REQUEST: iTunes Script access from menu bar

Kind feature request
Product Synergy
When Created 2004-12-30T08:44:53Z, updated 2006-07-05T02:03:53Z
Status open
Reporter bugzilla user 62
Tags no tags

Description

A simple way to dramatically increase the power of Synergy would be to allow access to iTunes's scripts from the Synergy menu bar interface. The menu that appears when the play/pause button is right-clicked could include an item "Scripts" (or even the script logo), and then that item could open a submenu that lists all scripts listed in the ~/Library/iTunes/Scripts folder. Thanks.
