Feature request #285: REQUEST: Hot key to start playing from user-definable Playlist

Kind feature request
Product Synergy
When Created 2005-08-08T06:00:43Z, updated 2007-11-29T02:30:48Z
Status open
Reporter bugzilla user 238
Tags no tags

Description

I would love to be able to set a particular hotkey to select and start playing a particular playlist. It would be fine by me if this was a pre-selected playlist (in the preference pane) or even the last playlist used. The problem is that whenever I plug in my iPod to update/charge it, the iPod playlist is automatically selected even though it is not playable. I am almost always playing the "Party Shuffle" now, but frequently the hotkeys become a bit useless (through no fault of yours) because an unplayable playlist is selected.

This would make me extremely happy and I might even consider purchasing Synergy Advance if it has this feature.
