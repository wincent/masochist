Feature request #384: REQUEST: Ability to change genres

Kind feature request
Product Synergy
When Created 2006-03-22T20:24:49Z, updated 2006-07-05T02:07:35Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From this forum thread:

http://wincent.com/a/support/forums/showflat.php?Number=948

"I would love to have a feature where I can change genres through Synergy. Similar to changing the rating for a song, though I guess you’d need a list of available genres already in iTunes when the shortcut is invoked. However, it would make so much easier."

Thanks for submitting the feature request. Will just need to decide how best to implement this:

1. Selection from submenu in Global Menu.

2. Hot key to cycle (forward/back) through genres

3. Type-ahead selection of the genre

4. Mouse-based point-and-click (ie. would require buttons or pop-up menu)

5. Right-click on floater to bring up contextual menu
