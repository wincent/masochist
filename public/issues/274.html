Feature request #274: REQUEST: Click to dismiss Floater

Kind feature request
Product Synergy Advance
When Created 2005-07-26T06:12:33Z, updated 2006-07-05T02:06:14Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

By default the Floater is transparent to mouse clicks. It would be nice if an option were provided to turn this behaviour off, and be able to dismiss the Floater on clicking it with the mouse. From a user email:

"click on it, it immediately disappears -- very useful when the floater appears on top of something you're currently using."

Comments

  1. Greg Hurrell 2005-07-30T18:55:11Z

    See also request #282.

    One idea is to make the floater behave like a Dashboard widget and when the mouse is over it display a close gadget. In the case of Dashboard you have to hold the option key to see the widget unless you're already in add/remove mode. Synergy Advance could either replicate that behaviour (require option) or just show the gadget automatically whenever the mouse is over the floater.

  2. Greg Hurrell 2005-07-31T02:48:51Z

    I was discussing this idea with a user who has the floater set to "always" display. His suggestion was that if the floater is set to "always" display AND the user makes use of the "click to dismiss" feature then the floater should return whenever the next track starts.

    The other alternative is to have it return after a user-specifiable interval (like 5 or 10 seconds, for example).

    The third option is to have a "dismiss" have a permanent effect (ie. the floater will never return unless the user explicitly causes it to be displayed by pressing a hot key, selecting a menu item, or relaunching Synergy Advance).

    I see no barriers to have all of these possible methods made available as user-settable preferences.
