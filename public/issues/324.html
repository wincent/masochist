  2. Greg Hurrell 2006-05-15T20:37:26Z

    I am a bit of a doofus because I forgot to mention that the "Floater on the desktop" feature is already implemented and has been for a long time. It's just that there's no user interface for it.

    See the release notes for version 1.3 (September 2004!):

    http://wincent.com/a/products/synergy-classic/history/

    Specifically: 'New "power user" feature (accessible from command-line), display floater on desktop. To activate, enter "defaults write org.wincent.Synergy desktopFloater -bool true" in the Terminal and restart Synergy Classic. To deactivate, enter "defaults write org.wincent.Synergy desktopFloater -bool false".'
