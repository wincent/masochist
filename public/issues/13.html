Feature request #13: REQUEST: Automatically deactivate floater when iTunes is paused/stopped

Kind feature request
Product Synergy
When Created 2004-11-10T10:33:04Z, updated 2006-07-05T02:02:53Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email: "Because of the problem with hiding/showing the floater when i'm using a fullscreen app like VLC:

It would be very nice to have an option "Hide floater when iTunes is not playing" or something like that (eg when I paused iTunes)"

Note that it is already possible to turn the Floater on and off using a hot key, or from the Synergy Global Menu. Even so, I agree that it would be nice to have this as an additional feature, perhaps under the "Advanced" preferences.
