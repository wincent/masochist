Feature request #328: REQUEST: Hot key to add currently playing track to a playlist

Kind feature request
Product Synergy
When Created 2005-11-14T12:54:21Z, updated 2006-07-05T02:06:52Z
Status open
Reporter Robert D
Tags no tags

Description

Hey!

First of all, your product rocks and yadda yadda.. I usually never buy shareware but I've bought, used, and heavily enjoyed your product since version [... incoherant mutter ...].

Anyway, I have a bunch of Mp3s that didn't rip properly, and I think it'd be really cool if I could bind a key to add the currently playing song to a playlist so I can take a look later.

Sort of like an iPod's on-the-go playlist type thing.. I donno, it'd probably be useful for other people as well.

Think about it and look into it if you could. It's not really that big a deal, synergy is such a great app there's not really much you can do to make it better as far as I'm concerned.

Keep it up, I love this program! -Bob
