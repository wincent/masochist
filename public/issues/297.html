Bug #297: Synergy Inoperative with iTunes 5.0

Kind bug
Product Synergy
When Created 2005-09-11T15:24:31Z, updated 2005-10-19T22:00:25Z
Status closed
Reporter David Shapiro
Tags no tags

Description

Synergy 1.7 and 2.1 unresponsive since updating to iTunes 5.0 (no problems with previous versions); iTunes remains operative but attempting to access Synergy menu bar results in "spinning beachball".

Comments

  1. Greg Hurrell 2005-09-11T21:40:51Z

    I haven't had any problems with Synergy and iTunes 5.0 (Synergy 2.1 running on Mac OS X 10.4.2). There are some troubleshooting things you can try out that are described in this article here:

    http://wincent.com/a/knowledge-base/archives/2004/11/how_to_file_a_g.php

    Things like deleting your preferences file (which may be corrupt), ruling out other possible causes etc). It also mentions some ways of gathering useful information for solving problems and squishing bugs.

    We'll leave this bug report open so that other users can add comments to it. But as far as I know, Apple didn't make any substantive changes to iTunes' AppleScript implementation that would break backwards compatibility with any program or script that communicates with iTunes.

  2. Greg Hurrell 2005-10-19T22:00:25Z

    I'm going to mark this bug as CLOSED now that Apple has released iTunes 6.0. My recommendation is always to run the latest versions of iTunes, Synergy and Mac OS X that you can.
