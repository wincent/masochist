Feature request #194: REQUEST: Hot key to toggle Global Menu

Kind feature request
Product Synergy Advance
When Created 2005-04-20T01:33:51Z, updated 2005-05-03T06:24:08Z
Status closed
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

I'd like to make a feature request. I'd like to toggle the display of the separate global menu item with a hot-key combination and have it remember its last state, displayed or not displayed regardless of whether iTunes is running or not. I'm not displaying any other "controls" for iTunes in the menu bar, only the Synergy menu item. I'd like to be able to toggle the display of that item with a hot-key combination so that if I can display it or not when I wish.

Presently, I can almost accomplish this by only displaying the controls (and not the separate global menu item) and then toggling the display of the controls off and on with the hot-key combination. However, this requires that I not check the box "Display controls only when iTunes is running." This causes the controls to be displayed when I restart the Mac even if I had them hidden when I shut down.
