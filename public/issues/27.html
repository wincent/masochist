Feature request #27: REQUEST: More compact/tight layout of elements in floater

Kind feature request
Product Synergy
When Created 2004-11-18T02:00:30Z, updated 2006-07-05T02:03:01Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email: "a smaller tighter display area... I like to leave it running at the bottom corner of my screen throughout the day and it becomes half covered by the documents I'm working on (I'm a print designer) and so I can't see the song title or the top of the album cover. So, If the text could be closer together and offer a way to make the album cover size anywhere from 25-50% smaller than the smallest available multi-line mode... the one line mode is just too small and too long."

I am filing this as a separate bug request, although it really is just one of a number of potential requests for greater control and customizability of the Floater appearance. I am working on that for Synergy Advance, but filing this bug report against Synergy because it's a goal for that project too.
