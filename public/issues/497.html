Feature request #497: REQUEST: Growl notification only on Play (not on Pause)

Kind feature request
Product Synergy
When Created 2006-11-09T08:16:42Z, updated 2007-05-15T12:22:02Z
Status closed
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

"Could you make a slight change in the way it interacts with Growl? Rather than have Growl bring up a notification when iTunes both plays and pauses, could you give us the option to have Growl only display on the play command and not on pause? This is, after all, the default behavior of Synergy's own notifications. The pause display in Growl is redundant and a little annoying."
