Feature request #559: REQUEST: Auto-hide floating if iTunes paused/stopped for a long time

Kind feature request
Product Synergy
When Created 2007-04-06T10:49:00Z, updated 2007-03-31T11:00:44Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

" My floater window is set to show always. It does. It more than does. When the song is stopped and nothing is playing the same album cover and title of the last song is still there. I'd have to quit iTunes, I suppose, to remove it.

How about a setting where the floater will disappear after a set number of minutes (1? 5?) of no song activity? I would choose "1" or even less. I am constantly pressing my Hot Key to remove the floater until the next time I choose an album. But I really do prefer to have the display always there when I am listening."

So this something like an option labelled "Hide Floater if stopped/paused for more than X seconds" (where X would be customizable by the user).

I just need to figure out a way to put this in the user interface that will not be confusing to the user.
