From a user email:

I ran into a slightly surprising issue when using rvm. Obviously, before calling 'rake make', one should "rvm use system" to ensure you're not in a different rvm environment. It might be nice to put this in your docs:

+  rvm use system       # <-- if you're using rvm
   rake make
