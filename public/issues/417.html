Feature request #417: REQUEST: Larger size range for album art display

Kind feature request
Product Synergy

Status open
Reporter Greg Hurrell
Tags no tags

Description

From the forums:

"I'd also like the album art display to be a little bigger. I suppose ideally that would be something that can be set by the user."

http://wincent.com/a/support/forums/showflat.php?Number=982

So really this request has two parts: (1) The ability for the user to directly control the size of album art (independently of the size of the Floater itself or the font used in the floater; and (2) That the range of possible sizes be extended to allow for bigger album art.
