Bug #137: Some user pictures appear shrunken inside the menu

Kind bug
Product WinSwitch
When Created 2005-03-11T00:18:08Z, updated 2005-04-25T22:42:46Z
Status closed
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

"Sometimes, user pictures are displayed at the wrong size. Note my icon in the WinSwitch screenshot, and how it's smaller than the others - it appears to be 26x26 (?), while the chosen size is set to 32x32 in the prefs. The source image is the same dimensions as the others, as indicated by the Accounts prefpane.

AND: Again, this was not an issue with WinSwitch 2.0. All user pictures were of the same (correct) size in the menu there. So, I'm curious what may have changed here."

My reply:

Can you send me the image file which is getting shrunk so that I can do some testing here? Check in NetInfo Manager (/Applications/Utilities/NetInfo Manager.app) to see what image is actually being used, and send me that one. Sometimes Mac OS X uses a cached copy at a location like "/Library/ Caches/com.apple.user501pictureCache.tiff" even when you chose an image at a different path (for example, an image in your "Pictures" folder). To find out the path being used go to NetInfo Manager, look in "users", then the appropriate user, then the entry for "picture".
