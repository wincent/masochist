Feature request #669: Shuffle button in menu bar

Kind feature request
Product Synergy
When Created 2008-04-16T07:28:23Z, updated 2008-04-17T01:22:02Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

Just received a user request for this via email:

Is it technically possible to add the "SHUFFLE" control to your toolbar?

Comments

  1. Greg Hurrell 2008-04-16T07:30:56Z

    In the meantime, it's worth noting that there is one alternative way to control the shuffle setting without bringing iTunes to the front: you can set a hot key for toggling the shuffle mode (see the "Hot keys" tab of the Synergy preferences).

    In Synergy Advance I also have a menu item for that purpose (and I'm currently working on a new release to bring Leopard compatibility).

    Finally, the iTunes Dock menu itself has a toggle for the shuffle mode.

  2. Greg Hurrell 2008-04-17T01:22:02Z

    Oh, I should have mentioned that the menu-based access to the shuffle feature is not specific to Synergy Advance; if you look in the Synergy Global Menu you'll see it in the iTunes submenu as well.
