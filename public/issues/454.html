Bug #454: desktop background picture changes

Kind bug
Product Synergy
When Created 2006-08-28T20:12:22Z, updated 2006-08-30T10:25:31Z
Status open
Reporter paul
Tags no tags

Description

Whenever I change the picture on my desktop and then run itunes with Synergy running, the desktop picture changes back to a default picture in response to any program action (pause, play, next track, etc). It's anoying and somewhat troublesome. Please advise!

Comments

  1. Greg Hurrell 2006-08-30T10:25:31Z

    Not able to reproduce this one myself. Synergy only communicates with iTunes (via Apple Events). It doesn't attempt to talk to any other part of the operating system and it certainly doesn't try to meddle with the desktop picture.

    Have you tried some trouble-shooting as described here?

    http://wincent.com/s/bugs/

    You should try to produce the problem from another user account, see if it persists across reboots or logins, and rule out other third-party software which may be causing the problem. For what it's worth, it doesn't seem possible that Synergy could be causing this.
