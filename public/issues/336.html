Description

It seems that after Synergy Advance has displayed a bezel for "Extra visual feedback for Hot Keys" (in the Hotkeys prefpane), it causes some system-wide drawing issues at the spot in the screen where the feedback bezel appears.

Steps to reproduce: 1. Enable the pref 2. Invoke a hotkey, bringing up the feedback bezel 3. Launch Grapher.app (/Applications/Utilities/Grapher.app in Tiger) 4. From Grapher's Examples menu, choose chonchoid (or any 3d example)

What happens: Drawing errors in the same place the bezel appears
