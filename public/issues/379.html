Bug #379: Use iTunes album artwork instead of Amazon lookup

Kind bug
Product Synergy
When 2006-03-09T09:34:29Z
Status closed
Reporter Sveinbjorn Thordarson
Tags no tags

Description

Feature request:

I have been trying out your Synergy tool and I like it very much. However, I find that it tends to lag every once in a while when fetching album covers from Amazon.

For my part, I have carefully ID3 tagged all of my collection and added covers in iTunes -- and quite frankly, it would be much better if Synergy simply used the album artwork displayed by iTunes instead of fetching it from Amazon. This would also prevent dozens of megabytes worth of tiff files from accumulating in the Synergy application support folder.

Just a thought.

Cheers,

Sveinbjorn Thordarson
