Feature request #358: REQUEST: Skip within track definable amount

Kind feature request
Product Synergy
When 2006-02-01T05:50:44Z
Status closed
Reporter Jeppe Utzon
Tags no tags

Description

When listening to Podcasts or Audiobooks in iTunes, sometimes I miss a word or a sentence - either because of outside noise or because the voice is hard to understand. In those instances it would be nice to be able to skip back a specific amount of time, like 5 or 10 seconds. iTunes doesn't offer this other than using the mouse and if it's a long recording one often ends up too far back or by using the rewind button which isn't a very good solution either. So the request is for Synergy to offer skip forward and backward a user definable amount of time. Like the user could set the amount to 10 secs for instance, and then skip back and forth with key combos while listening to the story. It would also be nice for skipping forward if a song has a long or slow start.

Cheers, Jeppe

  1. Greg Hurrell 2006-02-01T06:27:38Z

    Thanks for submitting the request. Did you know that this feature is already available in Synergy Advance?

    http://wincent.com/a/products/synergy-advance/

    See "Skip forwards" and "Skip backwards" in the Hot Keys section:

    http://wincent.com/a/products/synergy-advance/help/preferences/hot-keys/

    You can also define the skipping interval with the "Skip forward/back interval" setting in the "Advanced" section of the preferences. The default is 3 seconds; you can set it anywhere between 1 and 60 seconds.

    http://wincent.com/a/products/synergy-advance/help/preferences/advanced/

    Further more, both Synergy and Synergy Advance allow you to rewind or fast forward within tracks by pressing and holding the hot keys for previous and next track (as opposed to just doing a press-and-release). In Synergy you can do the same by clicking-and-holding on the next/previous menu bar control buttons. Once I've added menu bar control buttons to Synergy Advance the same will apply.

    Given that the feature you ask for is already available in Synergy Advance and that there is room for new hot keys in the Synergy user interface I think it's fairly unlikely that I'll implement this feature. I'm going to mark this as WONTFIX but will leave the request open for comments so that if enough people show interest in it I will reconsider.
