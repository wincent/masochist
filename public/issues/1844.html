  9. rotorglow 2011-12-07T22:23:39Z

    Seems to be related to running iTunes 10.5.1 in 64-bit mode. Cover art stored in tracks will correctly display if you open iTunes in 32-bit mode (even if the machine itself is running the 64-bit kernel).

    Since iTunes 10.5 runs as a 64-bit app on Lion by default, it appears as broken for everyone. (This broke a number of visualizer plugins, incidentally, so Synergy isn't the only program to run into this problem.)

    Hope that help
