Feature request #59: Show clickable rating bar in menu bar

Kind feature request
Product Synergy
When Created 2004-12-08T10:12:53Z, updated 2009-10-17T09:04:00Z
Status open
Reporter Kerry Echols
Tags no tags

Description

I'd like to be able to have the song rating display in the menu bar to the left of the "previous track" button. I'd also like to be able to change the rating of a song, including giving it no rating if it already has one, from the menu bar by clicking on the 0 to 5 stars. This would save a lot of time if one is evaluating many songs one hasn't listed to in a while, and wishes to alter their respective ratings.

Also, I first went to the page where I could query to see if this had already been requested, but found it to be the single most complicated page of gobbledygook I've ever encountered (and I maintail all of our company's Macs), so sorry if this is a duplicate request, etc.
