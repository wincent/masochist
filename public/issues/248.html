Feature request #248: REQUEST: Better Growl support

Kind feature request
Product Synergy Advance
When Created 2005-05-25T02:43:51Z, updated 2005-07-31T03:14:04Z
Status closed
Reporter bugzilla user 200
Tags no tags

Description

It would be nice if within Synergy, I could choose which data are displayed in Growl notifications. Also, within Growl, it would be nice to have different notifications available, so that instead of "iTunes Update", I could choose settings for "new track," "paused track," "changed rating," and whatever else there might be.

Thanks!

Comments

  1. Greg Hurrell 2005-05-25T03:49:30Z

    Thanks for submitting the request. For now Synergy only has the one notification, "iTunes update". Synergy Advance has three at the moment which you can turn on and off in the Growl preference pane:

    1. "iTunes track change" 2. "iTunes player state change" 3. "iTunes information (hot key)"

    The first is shown whenever the track or information about the track changes. The second of these is shown on player state changes (like pausing, playing, stopping etc). The third is shown when the user presses the "Show Growl Notification" hot key. It would be fairly easy to add more notification types like this to Synergy.

    I also have a Growl preferences pane for Synergy Advance which will allow you to specify what gets shown in the notification. You can specify a format string to control exactly what gets included. That is one level of control that I can't add to Synergy because of UI cramp; there's simply nowhere to put it.

  2. Greg Hurrell 2005-07-31T03:10:16Z

    Synergy Advance 0.2 is out so if you want to check out the new Growl preference pane you can download the public preview release from here:

    http://synergyadvance.com/

    I'm going to reassign the bug to Synergy Advance due to the UI cramp problem I've already mentioned as this level of control is not going to make it into the UI for Synergy Classic.

  3. Greg Hurrell 2005-07-31T03:14:04Z

    Marking this one as CLOSED, but do feel free to add more comments if you think there need to be more options added to the preference pane.

    I am not sure how much interest there is to add more than the three classes of notification that SA already sends to Growl. The principle problem with this request is that having too many classes will quickly clutter the notifications listing in the Growl preferences (and here I refer to the *Growl* preference pane itself and not the Growl part of the *Synergy Advance* preferences) and I can imagine getting a few complaints if I add too much. If people are interested in having more separate classes added please post a comment here.
