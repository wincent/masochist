Bug #333: Player position within track displayed in floater window

Kind bug
Product Synergy
When 2005-11-20T18:21:01Z
Status closed
Reporter Guido
Tags no tags

Description

I often wonder how much is left in the currently playing iTunes track. It would be great if I could just bring up the Synergy floater window and, in addition to the usual track information, see the player position in the current track. You can already see the track duration in the window, so just a simple addition of the time into the track is all that that would be needed, something like "1:03/2:00." No graphical representation would even be necessary. Possible? Practical?
