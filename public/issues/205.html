Feature request #205: REQUEST: Preferences for Playlists, Genres and Artists submenus

Kind feature request
Product Synergy Advance
When Created 2005-05-03T03:57:13Z, updated 2006-07-05T02:05:16Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

Users with really huge libraries may wish to disable these submenus or limit the number of levels in them to save system resources. The number of items can easily grow to be quite huge because the same track can appear in:

Playlists -> Library Playlists -> (Other playlists) Genres -> All genres Genres -> (genre) -> All artists Genres -> (genre) -> (artist) -> All albums Genres -> (genre) -> (artist) -> (album) Artists -> All albums Artists -> (album)
