Bug #592: Keyboard shortcuts become disabled

Kind bug
Product Synergy
When Created 2007-07-06T10:16:42Z, updated 2007-07-07T12:29:38Z
Status open
Reporter Morley Chalmers
Tags no tags

Description

I'm using keyboard shortcuts to increase or decrease the "star" rating for pieces of music. Mostly it works, but occasionally it fails. The problem has been around for a while (at least three months) and continues in 3.1.3.

I can solve the problem by quitting Synergy and then restarting. No need to reboot the Mac.

There's no pattern to which keyboard shortcuts become disabled. For instance, I use Command-F13 to start and stop music, Shift-Command-> to increase star value, Shift-Command-< to decrease star value. This morning the increase command failed but the start and stop music command still functioned. There have been other occasions when the reverse has been the case.

I'm using Mac OS 10.4.10.

I'm willing to run some tests, if that would help.

Kind regards,

Morley Chalmers
