Feature request #219: REQUEST: Increment play count only if track has been playing for a certain time

Kind feature request
Product Synergy Advance
When Created 2005-05-04T05:18:26Z, updated 2006-07-05T02:05:28Z
Status open
Reporter William Styler
Tags no tags

Description

Hi there! Big synergy fan, already registered advance.

The "Increase Play count with skip" is a great feature, but I'd ideally like to see "Increase play count with skip (if track has played more than X seconds). That way if I'm just skipping through tracks randomly, their counts won't each get advanced, but if I listen to one for a while and skip to a different one, the count would be advanced. 15-30 seconds delay on increasing playcount would be great!

Thanks!

Will
