Feature request #42: REQUEST: Export current track info to text file

Kind feature request
Product Synergy
When Created 2004-11-25T05:16:20Z, updated 2006-07-05T02:03:20Z
Status open
Reporter Eve Atley
Tags no tags

Description

REQUEST: Freshly Squeezed Software makes a freeware app called RecentTunes (http://freshsqueeze.com/products/freeware/) which allows you to upload a recently played track in iTunes into a file, which then allows you to via PHP add this info to a blog or what have you. Seems to me that Synergy has a very similar feature set. It would be great to have the ability to have Synergy output the last played track to an external file, or turned off in prefs.
