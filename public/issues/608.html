Bug #608: Floater under all windows

Kind bug
Product Synergy
When 2007-10-31T07:03:57Z
Status closed
Reporter bugzilla user 480
Tags no tags

Description

Hi there

i have another feature request for Synergy. Because i have set up a multi monitor system i have activated to see the Synergy Floater all the time.

Unfortunately the floater window is on top of all other windows. So when i move an application over the Floater is still see the floater above the application.

Can we have an option to toggle the floater the be under all other windows?

Best regards Alex

Comments

  1. Greg Hurrell 2007-10-31T07:30:54Z

    This feature is already implemented, although it's a bit of a hidden feature. You can find the notes for it in the release notes for version 1.3:

    http://wincent.com/a/products/synergy-classic/history/

    Specifically:

    [Feature] New "power user" feature (accessible from command-line), display floater on desktop. To activate, enter "defaults write org.wincent.Synergy desktopFloater -bool true" in the Terminal and restart Synergy. To deactivate, enter "defaults write org.wincent.Synergy desktopFloater -bool false".

    GUI-level options for adjusting the floater level have been requested in bug #324, so I am going to close this one and mark it as a duplicate:

    http://wincent.com/a/support/bugs/show_bug.cgi?id=324
