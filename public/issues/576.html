Feature request #576: REQUEST: Display lyrics of current song in a sticky growl notification

Kind feature request
Product Synergy
When Created 2007-05-22T20:21:22Z, updated 2007-10-28T04:26:19Z
Status open
Reporter bastian
Tags no tags

Description

That feature will be added in the menubar (when you control+click the controls), displaying a sticky growl notification with the lyrics (searching the lyrics if they don't are in iTunes, or with options in the pref. pane, to turn on the lyric searching or iTunes song's lyrics)

That will be great, because if you have work of you are occupied you can't be reading the lyrics in the iTunes info window, so a sticky notification will be useful to make another thing with the currrent lyrics waiting for you.
