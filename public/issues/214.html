Bug #214: Animated dock icon doesn't work

Kind bug
Product Synergy Advance
When Created 2005-05-03T11:12:10Z, updated 2005-07-27T01:30:43Z
Status closed
Reporter Will Adler
Tags no tags

Description

The dock icon does not animate, as a preference would suggest.

Comments

  1. Greg Hurrell 2005-05-03T18:17:32Z

    You have to turn it on from the command line for now because the preference isn't exposed in the UI yet. This and a few other options appear in the "General" preferences pane, which will ship in the next preview.

    1. Quit the app.

    2. In the Terminal issue this command:

    defaults write com.wincent.SynergyAdvance AnimateDockIcon -bool YES

    3. Restart the app.

    You can repeat the process, substituting NO for YES to turn the animation off.

  2. Greg Hurrell 2005-07-27T01:30:43Z

    Public preview release 0.2 is now out on the servers, which adds a GUI option to control dock icon animation (among other things).
