Feature request #83: REQUEST: Double-clickable album art on desktop with drop shadow, no bezel

Kind feature request
Product Synergy Advance
When Created 2004-12-22T22:49:32Z, updated 2006-07-05T02:03:47Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email: 'Feature Request (searched bugzilla): Option to dynamically display iTunes album art on the desktop for the currently playing song (similar to "Clutter", but art changes) throughout the duration of the song. Art should feature a drop shadow, and should not appear in a frame or bezel. I tried a few Konfabulator Widgets, but many are buggy and add needless features (such as simulated jewel cases). Similar to Clutter, clicking on the art icon could play/pause the song. Album art would disappear if iTunes is not running. Mouse over shows song title, etc.

I realize that this feature slightly overlaps the floater feature; however, I would prefer to see the Album Art longer than a floater allows, without it getting in the way.'

Thanks for the suggestion. This sounds like something that would be good for Synergy Advance.

In the meantime, the closest Synergy can offer is the ability to display the floater at desktop (icon level). See the changelog for Synergy 1.3 (the first version to introduce this feature):

http://wincent.com/a/products/synergy-classic/history/

Combined with the ability to display the floater permanently, this is as close as Synergy can currently get to what you're asking for.

There is some overlap with the feature you're requesting and the already filed bug #26, but I think it's different enough to warrant a separate entry in the database. Once the request recorded in bug #26 is fulfilled, then all that's needed is the removal of the bezel (perhaps that should be an additional option under that bug), and then the double-click to play/pause functionality.
